# GoGaffa Brand

> The World Cup just got personal.

## Quick Reference

See `BRAND_GUIDELINES.md` for the comprehensive brand guide, and `tokens.json` for machine-readable design tokens.

Website: [gogaffa.com](https://gogaffa.com)

## Palette

| Name      | Hex       | Role                        |
|-----------|-----------|-----------------------------|
| Cream     | `#f5f0e8` | Background / paper surface  |
| Ink       | `#1a1a2e` | Primary text, borders       |
| Red       | `#e63946` | Energy, CTAs, urgency       |
| Blue      | `#1d3557` | Identity, trust, links      |
| Purple    | `#6b4c9a` | Tournament, prestige        |
| Success   | `#2f7a4d` | Confirmation, positive      |

## Typography

- **Caveat** — handwritten personality (headlines, buttons, card copy)
- **Inter** — system UI clarity (body text, eyebrows, metadata)

## Voice

- Personality: playful, competitive, direct, football-native, inclusive banter
- Avoid official FIFA affiliation language
- Prefer terms: tournament card, footballer card, fan card, squad card, group, bounty, Locker Room
- Use "mates" not "friends" or "users"
- Sentence case everywhere (except brand name "GoGaffa")
